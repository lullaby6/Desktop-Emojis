# Desktop Emojis

An bipolar emoji on the bottom-right side of your screen that accompanies you.

Made with pygame and win32 apis.

## Requeriments

- ```Windows```
- ```Python```

## Run

Clone or download this repository and open the file ```main.pyw``` or run the command:

```bash
python main.pyw
```
